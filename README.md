 Responsive Email Template

This repository contains a fully responsive HTML email template designed for ABSA. The template is optimized for a wide range of devices and email clients, ensuring consistent rendering across desktops, tablets, and mobile devices.

Features

Responsive Design: Adapts seamlessly to different screen sizes using fluid layouts and media queries.

Cross-Client Compatibility: Tested on major email clients including Outlook, Gmail, Apple Mail, and Yahoo Mail.

Customizable Sections: Easily update text, images, buttons, and links to match branding or campaign requirements.

Inline CSS: All styles are inlined for maximum compatibility across email clients.

Call-to-Action (CTA) Buttons: Styled buttons that work across all supported email clients.

Accessibility Considerations: Includes proper semantic HTML and contrast for readability.
